Summary

The object of the exercise was to analyze changes in weather patterns as one get closer to the equator. This was accomplished by accessing data from OpenWeatherMap, which allowed access to a dataset of more than 500 cities.

The data was then analyzed using pandas, them plotted using Matpltlib, that is, Matplotlib was used to graph latitude to the four variables, windspeed, max temperature, cloudiness, and humidity.
